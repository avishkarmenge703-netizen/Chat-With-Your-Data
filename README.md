# Chat with Your Data

A professional web application that lets you upload CSV/Excel files and ask questions about your data in natural language. Powered by LangChain and OpenAI.

## Features

- 📂 Upload CSV or Excel files
- 💬 Chat interface to ask questions
- 🔍 Data preview with basic statistics
- 🧠 AI-powered answers using GPT-3.5
- 🎨 Clean, modern UI
- 🚀 Free deployment on Streamlit Cloud

### Prerequisites

- Python 3.8 or higher
- OpenAI API key (get one at [OpenAI Platform](https://platform.openai.com/api-keys))
